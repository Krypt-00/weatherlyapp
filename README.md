# 🌤️ Weather App

Web application that displays current weather and real-time clock, with **automatic location detection** and **responsive design**. Perfect for travelers, developers learning APIs, or as a portfolio project.

![Weather App Demo](https://img.shields.io/badge/demo-online-green) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow) ![OpenWeatherMap](https://img.shields.io/badge/API-OpenWeatherMap-blue)

## ✨ Features

- 🕐 **Real-time clock** that updates every second
- 🌍 **Automatic geolocation** to show weather in your region
- 🔍 **City search** with support for thousands of locations
- 🌡️ **Complete weather information**: temperature, feels like, humidity, wind speed
- 🎨 **Modern design** with glassmorphism effects and smooth animations
- 📱 **Fully responsive** - Works perfectly on mobile, tablet, and desktop
- ⚡ **Fast and lightweight** - No heavy frameworks, pure JavaScript

## 📋 Prerequisites

- Modern web browser
- Internet connection
- [OpenWeatherMap](https://openweathermap.org/api) API key (free)

This was coded in spanish so you'll see the name of functions or ids in spanish.
